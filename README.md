## Çalıştırmak için

Proje klasöründe, komutu çalıştırın:

### `npm i`
### `npm start`

Uygulamayı geliştirme modunda çalıştırır.\
[http://localhost:3000](http://localhost:3000) yolunda tarayıcınızda çalıştırır.

Değişiklik yaptığınızda sayfa yeniden yüklenecek.\
